# ktv-v3
## About
This is the 3rd version of the ktv website made by Teck Yong. This website aims allow users to easily play songs from a playlist that other people can easily contribute to, ensuring that everyone in the party has an equal chance of the song of their choice to be played. 

## What's new
I will be adding new features that are missing from ktv-v2 such as:
- Mutiple playlists per account
- Email based account system
- Password protected rooms
- Remote control
- Allowing users to play music from other rooms locally
